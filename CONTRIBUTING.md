## Hướng dẫn cài đặt

Clone project:

```bash
git clone https://github.com/npv2k1/nstore-microservice.git
```

Cài đặt package ở các thư mục `api` và `web`

```
pnpm install
```

Để bắt đâu có thể sử dụng docker-compose

```bash
docker-compose up -d
```

Cấu trúc thư mục

```
apps
    api
    nginx
    web
docker
    nginx
```
